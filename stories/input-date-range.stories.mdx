# InputRangeDate - инпут для выбора даты

### Инпут для выбора диапазона даты , возвращает два объекта типа Date , работает на основе компонента **DatePicker** и **RangeDatePicker**

`import InputRangeDate from "@estelink/ui/Input/DatePicker/InputRangeDate.vue";`

### Принимаемые параметры

#### **_первоначальное значение нижней границы ,если значение не передано , то по умолчанию 1 января 1970го_**

` startValue?: Date ;`

#### **_первоначальное значение верхней границы границы ,если значение не передано , то по умолчанию 31 декабря 2050го_**

` endValue?: Date ;`

#### **_минимальная возможная дата_**

` min?: Date;`

#### **_максимальная возможная дата_**

` max?:Date;`

#### **_readonly_**

`readonly?: boolean;`

#### **_display_**

`display?: displays;`

### События

#### **Стандартный v-model для startValue**

`(e: "update:startValue", value: Date): void;`

#### **Стандартный v-model для endValue**

`(e: "update:endValue", value: Date): void;`

#### **_Событие нажатия на валидную дату в календаре, возвращает оба значения , срабатывает при смене верхней и нижней границы_**

` (e: "onChange", value:{start:Date,end:Date}): void;`

## **Готовый встроенный компонент выглядит так **

```html
<InputRangeDate
  class="w-[240px]"
  @onChange="({ start, end }) => inputRangeHandler(start, end)"
  :start-value="new Date(2023, 2, 1)"
  :end-value="new Date(2023, 5, 0)"
  :min="new Date(2000, 2, 1)"
  :max="new Date(2030, 5, 0)"
></InputRangeDate>
```

### Диапазон

2.2. Тип “День”. В нем автозаполнение:

2.2.1. “Вчера”
При выборе данного автозаполнения выбирается дата вчерашняя в обоих полях.
2.2.2. “Сегодня”
При выборе данного автозаполнения выбирается дата сегодняшняя в обоих полях.
2.2.3. “Завтра”
При выборе данного автозаполнения выбирается дата завтрашняя в обоих полях.

2.3. Тип “Неделя”. В нем автозаполнение:

2.3.1. “Прошлая неделя”
При выборе данного автозаполнения выбирается даты из прошлой недели, дата понедельника и воскресенья.

2.3.2. “Последние 7 дней”
При выборе данного автозаполнения выбирается даты: дата “до” - сегодняшняя, дата “от” семь дней назад, включая сегодня.

2.3.3. “С начала этой недели”
При выборе данного автозаполнения выбирается даты: дата “до” - сегодняшняя, дата “от” начало недели, в которой состоит сегодняшний день.

2.3.4. “Эта неделя”
При выборе данного автозаполнения выбирается даты начала (понедельник) и завершения (воскресение) “сегодняшней” даты в недели.

2.3.5. “До конца этой недели”
При выборе данного автозаполнения выбирается даты: дата “от” - сегодняшняя, дата “до” завершение недели, в которой состоит сегодняшний день.

2.3.6. “Следующая неделя”
При выборе данного автозаполнения выбирается даты следующей недели, которая следуют после даты “сегоднешнего” дня, дата начала (от) понедельник, дата завершения (до) воскресение.

2.4. Тип “Месяц”. В нем автозаполнение:

2.4.1. “Прошлый месяц”
При выборе данного автозаполнения выбирается даты прошлого месяца, относительно текущего: первое число и последние.

2.4.2. “Месяц”
При выборе данного автозаполнения выбирается даты: дата “до” - сегодняшняя, дата “от” в прошлом месяце минус 31 день от сегодняшней. По хорошему учесть, что в разных месяцах разное количество дней, соответственно 31 это не константа.

2.4.3. “С начала этого месяца”
При выборе данного автозаполнения выбирается даты: “до” - сегодняшняя, дата “от” первое число месяца, в котором состоит сегодняшний день.

2.4.4. “Этот месяц”
При выборе данного автозаполнения выбирается даты: дата “от” начала месяца, дата “до” конец месяца, в котором состоит сегодняшний день.

2.4.5. “До конца этого месяца”
При выборе данного автозаполнения выбирается даты: дата “от” сегодняшний день, дата “до” завершение месяца, в котором состоит сегодняшний день.

2.4.6. “Следующий месяц”
При выборе данного автозаполнения выбирается даты следующего месяца, относительно текущего: первое число и последние.

2.5. Тип “Год”. В нем автозаполнение:
2.5.1. “Прошлый год”
При выборе данного автозаполнения выбирается даты прошлого года, относительно текущего: первое число января и последние декабря.

2.5.2. “С начала этого года”
При выборе данного автозаполнения выбирается даты: “до” - сегодняшняя, дата “от” первое число января, в котором состоит сегодняшний день.

2.5.3. “Этот год”
При выборе данного автозаполнения выбирается даты: дата “от” первое число января, дата “до” последение число декабря, в котором состоит сегодняшний день.

2.5.4. “До конца этого года”
При выборе данного автозаполнения выбирается даты: дата “от” сегодняшний день, дата “до” последний день декабря, в котором состоит сегодняшний день.

2.5.5. “Следующий год”
При выборе данного автозаполнения выбирается даты следующего года, относительно текущего: первое число января и последние число декабря.
